# バドミントン分析アプリ（ダブルス用）

ダブルスの試合結果を記録し、Eloレーティングを用いて選手やペアの実力を分析するWebアプリです。

単なる勝率だけではなく、相手の強さを考慮したレーティングや相性分析を行うことで、より実践的な分析を目指しました。

---

## 主な機能

### 試合記録
- ダブルスの試合結果を登録
- 試合履歴の一覧表示
- 試合データの削除

### プレイヤー分析
- 個人Eloレーティング
- Eloランキング表示
- Elo推移グラフ
- 勝率表示
- 平均得失点差
- パフォーマンスの安定度分析
- 直近5試合の成績表示

### ペア分析
- ペアEloレーティング
- ペアランキング表示

### 相性分析
- 相手ペアごとの勝率
- Eloによる期待勝率との差を表示
- 得意・苦手な相手の可視化

### 勝率予測
- 過去の試合データを利用した試合結果予測
- 個人EloとペアEloを組み合わせて計算

---

---

## Eloレーティングについて

このアプリでは以下の要素を考慮してEloを計算しています。

- 初期レート：1500
- 試合数に応じたK値の変動
- 得点差によるレート変動補正
- 個人EloとペアEloを独立して管理

勝率予測では、

```text
予測レート = 0.7 × ペアElo + 0.3 × 個人Elo平均
```

として計算しています。

## 制作背景

強豪クラブで練習する中で、自身の成長を感覚だけでなく客観的に把握したいと考えるようになりました。

そこで、チェスなどでも利用されているEloレーティングを導入し、

- 誰が強いのか
- どのペアの相性が良いのか
- 次の試合ではどちらが有利なのか

を分析できるアプリを制作しました。

FlaskによるWebアプリ開発、SQLiteを用いたデータ管理、統計指標の設計・実装の学習も兼ねています。
